# Local Active Directory Simulator

## Overview

This project documents the design, implementation, and ongoing development of a home laboratory built to simulate an enterprise Microsoft Active Directory environment.

The primary objective is to gain practical experience with Windows Server administration, Active Directory Domain Services (AD DS), DNS, Group Policy, PowerShell, networking, and enterprise troubleshooting.

Rather than simply following tutorials, every component is built incrementally and documented to understand the reasoning behind each design decision.
